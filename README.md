<!-- GETTING STARTED -->

## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/NarSiiiS/rj-downloader.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your url in `.env.local`
   ```dotenv
   NEXT_PUBLIC_BASE_URL=https://rj-api.vercel.app
   ```
